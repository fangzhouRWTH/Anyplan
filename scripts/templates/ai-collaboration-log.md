# AI Collaboration Log

Chronological record of meaningful AI-assisted work. Small mechanical tasks may need only a final summary entry.

## Entry Template

```markdown
### YYYY-MM-DD — Short title

- **TaskBrief**: intent, scope, acceptance
- **Workflow**: default-collaboration-loop (stages used)
- **ContextSnapshot**: key files read, structure summary
- **Changed artifacts**: paths
- **Verification**: commands run and results
- **Open questions**: follow-ups
```
